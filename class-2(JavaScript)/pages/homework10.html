[#10] Array, for 반복문 실력향상! 2022. 07. 28

오늘은 전에 배웠던 Array와 for 반복문을 활용해서 문제를 풀어보았다. 입력한 값이 array 변수에 저장되어있는지를 확인하는 방법과 for 반복문을 두 번 사용해서 푸는 문제, array 값을 입력하면 평균값을 내서 기존 평균값과 비교를 해주는 문제를 풀어보았다. console 창에 출력하는 문제들이라 오늘은 따로 시각적인 화면을 만들지는 않았다. 이것도 재밌긴 하지만 역시 시각적인 화면이 있는게 좋은 것 같다..ㅎ 점점 많은 것들을 배우고 있어서 기억은 잘 안나지만 그래도 이해하면서 지나가서 앞으로도 찾아서 잘 써먹을 수 있을 것 같다!
