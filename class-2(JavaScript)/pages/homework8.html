[#8] 탭 메뉴 만들기 2022. 07. 25

드디어 해보고 싶었던 탭을 만들어보았다. 그동안 제이쿼리를 쓰지 않다가 for문을 쓰기 위해 선생님이 알려주신 방법대로 제이쿼리를 사용하여 코드를 작성해보았다. for문은 반복되는 코드를 작성해야할 때 한 번만 쓸 수 있는 구문이었는데, 반복되는 탭 버튼들과 탭에 들어가는 콘텐츠들에 classlist를 제거, 추가 하면서 탭 기능을 구현해보았다. 확장성 있는 코드를 위해 탭이 계속 추가되는 것을 감안하여 tab-btn의 길이(length)를 구하여 그것보다 작은 수 까지 반복문을 사용하도록 하여 확장성 있는 코드를 작성했다. var가 아닌 let 을 사용해야 코드가 작동하는데 그 이유는 클릭이벤트가 있는 경우 클릭 하지 않으면 클릭이벤트 내에 있는 코드는 실행되지 않고 반복문만 실행되는데, var 변수의 경우 function 전체가 범위이기 때문에 반복문을 돌리고 나면 i에는 0값이 아닌 설정해둔 반복문 범위까지의 값이 남기 때문에 이후에 0에 해당하는 버튼을 누르면 에러가 난다. 그렇기에 범위가 function 전체가 아닌 클릭 이벤트 내의 {}가 범위인 let을 설정해 해당 에러를 해결할 수 있다.
탭 메뉴를 만들어보는김에 곧 오픈될 제조도 화섬의 공인즙개사 실험일지 게시판 디자인을 가져와봤다. 내가 디자인 했던 화면을 내가 개발해보니까 진짜 신기하고 뿌듯하다.. 물론 이 게시판의 정상적인 서비스를 위해서 개발해야하는 부분이 훨씬 많지만 이걸 만들어낸 것 만으로도 뿌듯허다.. 히힛
